# Book API

This is a simple RESTful API for managing a collection of books. It allows users to create, read, update, and delete book records. The API is built using Node.js, Express, and MongoDB.

## Features
- Create a new book
- Get all books
- Get a book by ID
- Update a book by ID
- Delete a book by ID

## Technologies Used
- Node.js
- Express
- MongoDB
- Mongoose
